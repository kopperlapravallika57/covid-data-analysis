# COVID-19 Data Analysis Project

This is a **Python project** that analyzes COVID-19 data and visualizes total cases by country using **pandas** and **Matplotlib**.

## Features
- Display COVID-19 cases by country
- Bar chart visualization of total cases
- Easy to expand with more datasets
- Perfect for data science beginners

## Technologies Used
- Python
- pandas
- Matplotlib

## How to Run
1. Clone or download this repository
2. Make sure Python is installed
3. Install required libraries:
```bash
pip install pandas matplotlib
